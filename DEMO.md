# ROS2教学网站功能演示

## 🎯 网站概览

这是一个专为ROS2 Humble设计的交互式教学网站，集成了智能AI聊天助手，为学习者提供全方位的学习体验。

## 🚀 主要特性展示

### 1. 响应式导航栏
- 固定顶部导航，始终可见
- 渐变蓝紫色背景设计
- 平滑滚动到对应章节

### 2. ROS2教学内容模块

#### 入门指南
- ROS2简介和定位
- 详细的安装步骤
- 环境配置说明

#### 基础概念卡片
- **节点 (Nodes)** - 系统基本执行单元
- **话题 (Topics)** - 异步通信机制  
- **服务 (Services)** - 同步通信模式
- **动作 (Actions)** - 长期任务处理

#### 交互式教程
- 创建第一个节点
- 发布者和订阅者模式
- 服务和客户端实现
- 参数系统使用

#### 代码示例
- Python/C++双语言支持
- 语法高亮显示
- 标签页切换功能
- 完整可运行代码

### 3. AI聊天助手核心功能

#### 窗口特性
```
🔸 可拖拽移动 - 鼠标拖拽标题栏自由移动
🔸 最小化功能 - 点击"-"按钮收起/展开
🔸 关闭功能 - 点击"×"按钮完全关闭
🔸 模型选择 - 下拉选择不同AI模型
```

#### 智能问答
当前支持的ROS2话题：
- ROS2基础概念解释
- 节点、话题、服务详解
- 安装和配置指导
- 错误排查建议
- 代码示例说明

#### 交互体验
- 打字机效果的回复动画
- 加载状态显示
- 自动滚动到最新消息
- 友好的欢迎界面

### 4. 响应式设计

#### 桌面端 (1200px+)
- 聊天窗口350px宽，500px高
- 位于右下角，可自由拖拽
- 完整功能展示

#### 平板端 (768px - 1199px)  
- 自适应布局调整
- 导航栏响应式折叠
- 内容区域优化排列

#### 移动端 (< 768px)
- 聊天窗口全屏显示
- 触摸优化的交互
- 简化导航菜单

## 🛠️ 技术实现

### 前端技术栈
- **HTML5**: 语义化标签
- **CSS3**: Flexbox/Grid布局，动画效果
- **JavaScript ES6+**: 类、模块、异步处理
- **Highlight.js**: 代码语法高亮
- **响应式设计**: 媒体查询适配

### AI聊天实现
```javascript
class AIChat {
  // 拖拽功能实现
  initDragging() { ... }
  
  // 消息处理
  sendMessage() { ... }
  
  // 智能回复生成
  generateResponse() { ... }
}
```

### 模块化结构
- `main.js`: 页面核心功能
- `chat-widget.js`: AI聊天组件
- `styles.css`: 样式组件
- `config.json`: 配置管理

## 📱 使用场景演示

### 学习路径1: 新手入门
1. 访问网站首页
2. 阅读"入门指南"了解ROS2
